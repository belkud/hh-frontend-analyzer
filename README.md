# HH Analyzer

Анализатор востребованности технологий в программировании на основе вакансий с headhunters.

## Описание

Приложение собирает вакансии по вашему запросу (по умолчанию "Frontend") и через API hh.ru анализирует, какие технологии чаще всего упоминаются в требованиях. Результат выводится в виде рейтинга с процентным соотношением.

Поддерживается анализ **250+ наиболее популярных технологий**, включая:
- Фреймворки и библиотеки: React, Vue, Angular, Svelte и др.
- Языки: JavaScript, TypeScript, Dart и др.
- Инструменты сборки: Vite, Webpack, esbuild и др.
- CSS-фреймворки: Tailwind, Bootstrap, Material UI и др.
- State Management: Redux, Pinia, Vuex, Zustand и др.
- Тестирование: Jest, Vitest, Cypress, Playwright и др.

## Технологии

- **Vue 3** (Composition API, `<script setup>`)
- **TypeScript**
- **Vite**
- **Axios**

## Установка

```bash
npm install
```

## Запуск

```bash
# Режим разработки
npm run dev

# Сборка для продакшена
npm run build

# Предпросмотр сборки
npm run preview
```

## Как это работает

1. На hh.ru собираются 2000 самых 'свежих' на текущий день вакансий по ключевому слову (география — вся Россия)
2. Идёт анализ технологий
3. Вывод в консоль 20 наиболее популярных технологий
